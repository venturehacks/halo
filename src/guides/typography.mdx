## Line Height

> ** ⚠️ Advanced override. Use with caution.**

Generally, you should not override line height. There is a curated line height automatically applied when setting text size.

<Playground>
  <Box maxWidth="600px">
    <Paragraph>
      <strong>Default.</strong> The Paragraph component is ideal for rendering blocks of copy. It offers a narrower subset of features from Span, with stronger default enforcement of type size and line height. By default, it applies a natural document flow bottom margin.
    </Paragraph>
    <Paragraph lineHeight="comfortable">
      <strong>Comfortable.</strong> The Paragraph component is ideal for rendering blocks of copy. It offers a narrower subset of features from Span, with stronger default enforcement of type size and line height. By default, it applies a natural document flow bottom margin.
    </Paragraph>
    <Paragraph lineHeight="compact">
      <strong>Compact.</strong> The Paragraph component is ideal for rendering blocks of copy. It offers a narrower subset of features from Span, with stronger default enforcement of type size and line height. By default, it applies a natural document flow bottom margin.
    </Paragraph>
    <Paragraph lineHeight="dense">
      <strong>Dense.</strong> The Paragraph component is ideal for rendering blocks of copy. It offers a narrower subset of features from Span, with stronger default enforcement of type size and line height. By default, it applies a natural document flow bottom margin.
    </Paragraph>
  </Box>
</Playground>

```scss
$line-height--default: 1.5;
$line-height--comfortable: 1.25;
$line-height--compact: 1.125;
$line-height--dense: 1.0625;
$line-heights: (
  default: $line-height--default,
  comfortable: $line-height--comfortable,
  compact: $line-height--compact,
  dense: $line-height--dense,
);
```

## Sass

Configure many typeface properties at once. For each setting, it will only spit out a rule if you provide a non-null value.

If a supplied preset key name is invalid for `$family`, `$size`, `$line`, `$weight`, etc,
the mix- falls back on supplying the literal value to
the cooresponding rule.

```scss
@mixin font(
  $size: null,   // 2xs | xs | sm | md | lg | xl | 2xl | 3xl | 4xl | 5xl | 6xl
  $line: null,   // default | comfortable | compact | dense
  $weight: null, // ultralight | light | regular | medium | semibold | bold
  $color: null,  // palette color name or hex
  $family: null, // primary | serif | monospace | system
  $transform: null, // uppercase | lowercase | capitalize | none
  $antialiased: null,
  $spacing: null,
  $truncate: null
)
```
