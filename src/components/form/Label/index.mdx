# Label

Wrap an input field with this component to surround with an accessible `<label>`, complete with title and supporting text. Unlike the raw form components that must be wrapped or manually wired up to a form library, `Label` works out of the box with no requirment for a form-connected wrapper.

<Playground>
  <Box align="stretch" maxWidth="500px">
    <Label field="name" title="My field title" supportingText="Such supporting text">
      <RawInput />
    </Label>
  </Box>
</Playground>
