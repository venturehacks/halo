# Banner

Display a prominent message and optional action. Banners are displayed at the top of the screen and may be either persistent until dismissed or expired after some time. Avoid stacking more than one banner.


### Props

<Props of={Banner} />

### Exports

```tsx
export type BannerVariant =
  | 'default'  // Lowest priority
  | 'warning'  // Mid priority, "just so you know" type messages
  | 'error'    // High priority blocker
  | 'success'; // High priority success
```

### Usage

<Playground>
  <Box maxWidth="640px">
    <Header h4>Static, with inline links</Header>
    <Box margin={{ bottom: 4 }}>
      <Banner>
        Find more jobs. <Span regular><a href="#">Try clearing your filters</a>.</Span>
      </Banner>
      <Banner variant="warning">
        Draft Mode. <Span regular>To make public, <a href="#">publish this profile</a>.</Span>
      </Banner>
      <Banner variant="error">
        This profile is banned from Talent. <Span regular>They cannot recruit on AngelList.</Span>
      </Banner>
      <Banner variant="success">
        New password saved! <Span regular>Please <a href="#">log in</a> again.</Span>
      </Banner>
    </Box>

    <Header h4>Dismissable</Header>
    <Box margin={{ bottom: 4 }}>
      <Banner offerDismiss>
        Find more jobs. <Span regular>Try clearing your filters.</Span>
      </Banner>
      <Banner variant="warning" offerDismiss>
        Draft Mode. <Span regular>To make public, publish this profile</Span>
      </Banner>
      <Banner variant="error" offerDismiss>
        This profile is banned from Talent. <Span regular>They cannot recruit on AngelList.</Span>
      </Banner>
      <Banner variant="success" offerDismiss>
        New password saved! <Span regular>Please log in again</Span>
      </Banner>
    </Box>
  </Box>
</Playground>
