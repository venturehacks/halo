# Label

Wrap an input field or collection of fields (radios, checkboxes) in an accessible `<label>`.

Read more: [Label (MDN Web Docs)](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/label)

### Usage

<Playground>
  <div className="max-w-xl">
    <Label field="title" title="Title" byline="Wraps single input element">
      <RawInput />
    </Label>
    <Label field="iceCreamPreference" title="Ice cream preference" byline="Wrap collection of inputs that reference a single form field" containsFieldGroup>
      <div className="flex flex-col gap-4 px-1">
        <RawRadio label="Chocolate" checked />
        <RawRadio label="Vanilla" />
        <RawRadio label="Strawberry" />
      </div>
    </Label>
  </div>
</Playground>
