# RawSelect

Raw input fields are designed to be controlled by a [smart, form-connected parent](https://github.com/venturehacks/AngelList/blob/master/frontend/talent/components/Form/TextField/index.tsx).

### Usage

<Playground>
  <Label field="ice-cream" title="One True Ice Cream" byline="Select your favorite!" containsFieldGroup>
    <Box row align="space-between" className="max-w-lg">
      <RawSelect
        name="ice-cream"
        value="vanilla"
        options={[
          { label: 'Chocolate', value: 'chocolate' },
          { label: 'Vanilla', value: 'vanilla' },
          { label: 'Strawberry', value: 'strawberry' },
        ]}
        onChange={() => {}}
      />
      <RawSelect
        name="ice-cream"
        value="oops, error"
        options={[
          { label: 'Chocolate', value: 'chocolate' },
          { label: 'Vanilla', value: 'vanilla' },
          { label: 'Strawberry', value: 'strawberry' },
        ]}
        onChange={() => {}}
        hasError
      />
    </Box>
  </Label>
</Playground>

### Options

```jsx
import { RawSelect, RawSelectOption } from 'halo';

const options: RawSelectOption[] = [
  { label: 'Chocolate', value: 'chocolate' },
  { label: 'Vanilla', value: 'vanilla' },
  { label: 'Strawberry', value: 'strawberry' },
];
```
