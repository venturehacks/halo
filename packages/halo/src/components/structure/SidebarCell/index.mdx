# SidebarCell

Common pattern throughout Recruit products. Cell supports fixed title, byline, and arbitrary slots for header and chin (JSX children).

### Usage

<Playground>
  <div className="max-w-xs bg-blue-100">

      byline="California, Utah, Colorado, Remote"
      header="Last edit: 7 days ago"
      onClick={() => alert('onClick')}
      onStarClick={() => alert('onStarClick')}
      isActive
    />
    <SidebarCell
      title="Head of Product"
      byline="Europe, United States, Remote"

      isStarred
    />
