# Bracket

Component that isolates its first or last child from remaining children. Smart responsive defaults.

### Usage

<Playground>
  <Bracket className="max-w-xl">
    <Avatar
      size="lg"
      imageUrl="https://avatars1.githubusercontent.com/u/194885"
    />
    <span className="text-dark-a text-sm">Isolate first child (default)</span>
    <Button>Send message</Button>
  </Bracket>
  <hr />
  <Bracket isolate="last-child" className="max-w-xl">
    <Avatar
      size="lg"
      imageUrl="https://avatars1.githubusercontent.com/u/194885"
    />
    <span className="text-dark-a text-sm">Isolate last child</span>
    <Button>Send message</Button>
  </Bracket>
