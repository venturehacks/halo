# Grid

Declare a grid container `<Grid>` and fill with `<Cell>` content that snaps to a 24-column grid. Grid cells have sane responsive defaults and stack vertically on handset devices.

`<Cell>` is a thin preset on `<Grid>`<br />
`<Grid>` is a thin preset on `<Box>`

Both elements inherit most customizations offered by [`<Box>`](/structure/box) by forwarding props.
