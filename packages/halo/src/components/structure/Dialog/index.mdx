# Dialog

A UI element to confirm or cancel actions; easily embeddable within a modal.

### Usage

<Playground>
  <Dialog
    title="Are you sure you'd like to delete this message?"
    primaryButtonLabel="Delete"
    secondaryButtonLabel="Cancel"
  >
    This action cannot be undone.
  </Dialog>
</Playground>
