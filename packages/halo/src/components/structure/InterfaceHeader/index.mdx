# InterfaceHeader

Divides application sections into a uniform hierarchy. This component is deliberately inflexible, and customization is discouraged. Lacks embedded padding &amp; margin.

Compare to [Header](/core/header), which is appropriate for longform body copy.


### Usage

<Playground>
  <InterfaceHeader page>Page header</InterfaceHeader>
</Playground>

<Playground>
  <InterfaceHeader panel>Panel header</InterfaceHeader>
</Playground>

<Playground>
  <InterfaceHeader section>Section header</InterfaceHeader>
</Playground>

<Playground>
  <InterfaceHeader micro>Micro header</InterfaceHeader>
</Playground>
