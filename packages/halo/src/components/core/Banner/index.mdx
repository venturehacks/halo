# Banner

Display a prominent message and optional action. Banners are displayed at the top of the screen and may be either persistent until dismissed or expired after some time. Avoid stacking more than one banner.

### Usage

<Playground>
  <div>
    <InterfaceHeader section flow title="Static, with inline links" />
    <div className="mb-8">
      <Banner>
        Find more jobs. <span className="font-normal"><a href="#">Try clearing your filters</a>.</span>
      </Banner>
      <Banner variant="warning">
        Draft Mode. <span className="font-normal">To make public, <a href="#">publish this profile</a>.</span>
      </Banner>
      <Banner variant="error">
        This profile is banned from Talent. <span className="font-normal">They cannot recruit on AngelList.</span>
      </Banner>
      <Banner variant="success">
        New password saved! <span className="font-normal">Please <a href="#">log in</a> again.</span>
      </Banner>
    </div>

    <InterfaceHeader section flow title="Dismissable" />
     <div className="mb-8">
      <Banner offerDismiss>
        Find more jobs. <span className="font-normal">Try clearing your filters.</span>
      </Banner>
      <Banner variant="warning" offerDismiss>
        Draft Mode. <span className="font-normal">To make public, publish this profile</span>
      </Banner>
      <Banner variant="error" offerDismiss>
        This profile is banned from Talent. <span className="font-normal">They cannot recruit on AngelList.</span>
      </Banner>
      <Banner variant="success" offerDismiss>
        New password saved! <span className="font-normal">Please log in again</span>
      </Banner>
    </div>

  </div>
</Playground>
