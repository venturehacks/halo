# InterfaceHeader

Divides application sections into a uniform hierarchy. This component is deliberately inflexible, and customization is discouraged. Lacks embedded padding &amp; margin.

Compare to [Header](/core/header) (deprecated), which is appropriate for longform body copy.

### Usage

<Playground>
  <InterfaceHeader title="Page" byline="Connect to a payroll partner" page />
</Playground>

<Playground>

    title="Section"
    byline="Create a profile, post jobs, find candidates."
    section
  />
</Playground>
