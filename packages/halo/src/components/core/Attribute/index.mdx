# Attribute

Vertical key-value / term-definition. Produces [`<dt><dd>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dt) pair.

Be sure to wrap in `Attribute.List` to grant best semantics. Wrap is required for size adjustment.

### Usage

<Playground>
  <Attribute.List>
    <Attribute name="Term" value="Definition" />
    <Attribute name="Key" value="Value" />
  </Attribute.List>
</Playground>
