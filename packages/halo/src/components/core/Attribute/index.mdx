# Attribute

Vertical key-value / term-definition pair. Produces [`<dt><dd>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dt) pair.

Wrap in `AttributeList` for best semantics and ability to set group size.

### Usage

<Playground>
  <AttributeList>
    <Attribute name="Term" value="Definition" />
    <Attribute name="Key" value="Value" />
  </AttributeList>
</Playground>
