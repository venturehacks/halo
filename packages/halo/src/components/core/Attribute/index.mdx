# Attribute

Vertical key-value pair, great for job listing attributes. Produces [`<dt><dd>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dt) pair.

Wrap in `AttributeList` for best semantics, ability to set size.

### Usage

<Playground>
  <AttributeList>
    <Attribute name="AngelList" value="A platform for startups" />
    <Attribute name="Apple" value="Think different" />
  </AttributeList>
</Playground>
