```scss
  bold: $weight-bold,
);
```

## Sass

Configure many typeface properties at once. For each setting, it will only spit out a rule if you provide a non-null value.

If a supplied preset key name is invalid for `$family`, `$size`,`$weight`, etc,
the mix- falls back on supplying the literal value to
the cooresponding rule.

```scss
@mixin font(
  $size: null,
  // 2xs | xs | sm | md | lg | xl | 2xl | 3xl | 4xl | 5xl
  $weight: null,
  // regular | medium | semibold | bold
  $color: null,
  // palette color name or hex
  $family: null,
  // primary | serif | monospace | system
  $transform: null,
  // uppercase | lowercase | capitalize | none
  $spacing: null
);
```
