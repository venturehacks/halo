# Contributing a new Icon

<br/>

#### Download SVG

NOTE: the SVG icon should be 24x24.

<br/>

#### Copy SVG to Halo Icon Directory

```bash
# halo root
open packages/halo/src/components/icons
```

NOTE: filename should be pascal case, suffixed with 'Icon'

<img src='/add_icon_step_2.gif' />

<br/>

#### Regenerate Index

We want to index our new icon in the [directory](/icons), and declare it as an export of the halo module.

This chore has been automated:

```bash
yarn icons
```
<img src='/add_icon_step_3.gif' />
<br/>


#### Create Pull Request

[An example PR](https://github.com/venturehacks/halo/pull/154) that adds the `WarningIcon`, for reference.
