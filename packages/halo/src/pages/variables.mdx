# Variables

## Font Family

```scss
$font-family-primary: 'helvetica neue', arial, helvetica, freesans,
  'liberation sans', 'numbus sans l', sans-serif;
$font-family-serif: georgia, serif;
$font-family-monospace: Monaco, 'Courier New', monospace;
$font-family-system: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
  Helvetica, Arial, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji',
  'Segoe UI Symbol';

$font-families: (
  primary: $font-family-primary,
  serif: $font-family-serif,
  monospace: $font-family-monospace,
  system: $font-family-system,
);
```

## Weight

```scss
$weight-regular: 400;
$weight-medium: 500;
$weight-semibold: 600;
$weight-bold: 700;
$weight-heavy: 800;

$weights: (
  regular: $weight-regular,
  medium: $weight-medium,
  semibold: $weight-semibold,
  bold: $weight-bold,
  heavy: $weight-heavy,
);
```

## Line Height

```scss
$line-height-default: 1.5;
$line-height-comfortable: 1.25;
$line-height-compact: 1.125;
$line-height-dense: 1.0625;

$line-heights: (
  default: $line-height-default,
  comfortable: $line-height-comfortable,
  compact: $line-height-compact,
  dense: $line-height-dense,
);
```

## Text Transform

```scss

```
