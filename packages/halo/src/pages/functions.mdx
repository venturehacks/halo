# Functions

## tempo

Takes a magnitude parameter and multiplies it times the base Halo spacing unit `8px`. Magnitude cannot be more granular than increments of `0.5`.


```scss
@function tempo($magnitude) {
  @return 8px * $magnitude;
}
```

## grid-cell-width

Returns the width, in pixels, of an element that is N grid cells wide.

`1 <= $cells <= 24`

```scss
@function grid-cell-width($cells) {
  $cell-width: tempo(4); // 32px
  $gutter-width: tempo(2); // 16px
  $gutters: $cells - 1;
  @return ($cells * $cell-width) + ($gutters * $gutter-width);
}
```

## map-deep-get

```scss
@function map-deep-get($map, $keys...) {
  @each $key in $keys {
    $map: map-get($map, $key);
  }
  @return $map;
}
```

## has-unit

```scss
@function has-unit($v) {
  @return not unitless($v);
}
```

## strip-unit

```scss
// 24px => 24
@function strip-unit($value) {
  @return $value / ($value * 0 + 1);
}
```
